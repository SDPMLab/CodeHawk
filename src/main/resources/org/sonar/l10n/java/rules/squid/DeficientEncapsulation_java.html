This rule avoids Deficient Encapsulation

Noncompliant Example

When the declared accessibility of one or more members of an abstraction is more permissive than actually required. 

	public class DeficientEncapsulation {

		public int a = 1;

		public int a() {
			return a;
		}
	}

Compliant Solution

1.If this parameter is to be used by other classes, change its modifier to "not public" and generate "a public method" for other classes to obtain the parameter.
2.If this parameter is not to be used by other classes, then change its modifier "not public".
